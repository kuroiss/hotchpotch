使い方

最初にCSVファイルがあるディレクトリのパスを11行目に書いてください
相対パスでも通るしフルパスでも多分通ります

その状態で起動するとファイル一覧が左上に出ると思います
グラフを作成したいCSVファイルを選んで"select csv file"ボタンを押すと下の二つの空欄が要素で埋まると思います
各要素に名前を付けてない場合は一つしか表示されない可能性があるので、
その場合は名前を付けるかプログラムを書き換えてください

また、基本的にコンマ区切りのCSVファイルを想定しているので、
空白区切りのCSVファイルを読もうとすると要素が分割されなくてえらいことになります
空白区切りのやつを読みたい場合は、各所にある"pd.read_csv"関数にキーワード引数を与えてください
空白区切りにしたい場合は、read_csv関数の最後に
sep = '/s+'
という記述を入れると上手くいくと思います
だめだったらpandasのread_csvのリファレンスを読むかぐぐってください

要素がちゃんと表示されたら、左側で1つ、右側で好きな要素を選択して
'create graph overlap'もしくは'create graph separately'を押すとグラフが出ます
overlapは全部同じ所に表示、separatelyは各要素で別々のグラフ縦に並べて表示します

その下のxlog ylogのチェックボックスはそのままの意味です

その下の0と書いてあるテキストボックスは線の太さを決める所です
0だと散布図、1~だと折れ線になります

grid lineはまんまグリッド線の表示/非表示の切り替えです
x軸のみ、y軸のみに表示させたい人は頑張って書き換えてね
各所のax.gridを書き換えればうまくいけるんじゃないかな

最後のパスが書いてあるテキストボックスと"save all graph"ボタンは、
そのパスにx軸固定・y軸を各種要素に取ったグラフを1枚ずつ描画し、
自動で全部保存してくれるボタンです
パスは相対でも絶対でも通りますが、存在するディレクトリじゃないと怒られます
